# 权限说明

## 超级管理员
标记为 isAdmin 的用户为超级管理员，拥有系统所有权限

## 团队

- 创建者(owner) / 管理员(master) 拥有关联团队的所有操作权限 
    - 创建自己的团队
    - 修改团队
    - 获取团队列表
    - 创建项目
    - 获取团队成员列表
    - 删除成员
    - 修改成员角色
    - 添加成员
    - 离开团队
    - 查看团队动态
    
- 普通成员 
    - 创建自己的团队
    - 获取已加入的团队列表
    - 获取团队成员列表
    - 离开团队
    - 查看团队动态
